# Personal Profile Website

## Overview

This is a professional personal profile website developed as part of an assignment for SkillBridge Institute. The website showcases the portfolio of Girma Worku, a Full Stack Developer specializing in the MERN stack and an expert Data Scientist.

## Features

- **Home Page**: Introduction, profile image, interests, and services offered.
- **About Page**: Detailed background, education, skills, and professional experience.
- **Contact Page**: Contact form with fields for name, email, phone, and message, along with social media links.
- **Responsive Design**: Professional styling with a clean, modern layout using CSS.
- **Navigation**: Consistent navigation menu across all pages.

## Technologies Used

- **HTML5**: For structuring the content and forms.
- **CSS3**: For styling, layout, and responsive design.
- **Version Control**: Git and GitHub for repository management.

## Project Structure

```
personal-profile-website/
```

## Installation and Setup

1. **Clone the Repository**:
   ```
   git clone https://github.com/gworku/portfolioassigment_skillbridgeInstitute.git
   ```

2. **Navigate to the Project Directory**:
   ```
   cd personal-profile-website
   ```

3. **Open in Browser**:
   - Open `index.html` in your preferred web browser to view the website.
   - No server setup is required as this is a static HTML/CSS site.

## Usage

- Use the navigation menu to switch between pages.
- The contact form is functional for demonstration but does not send emails (HTML-only implementation).
- Social links are placeholders; replace with actual URLs as needed.
